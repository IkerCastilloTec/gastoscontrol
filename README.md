Sistema GastosControl
Autor

Iker Castillo

Descripción

Este proyecto es un avance de un sistema para el control de gastos diarios desarrollado en Java. Permite registrar gastos y llevar un control básico del dinero utilizado. Está organizado por paquetes para aplicar principios de programación orientada a objetos.

Funcionalidades implementadas

Agregar un gasto (descripción y monto)

Mostrar la lista de gastos registrados

Calcular el total gastado

Menú interactivo en consola

Estructura del proyecto

com.iker.gastoscontrol

Main.java

model/Gasto.java

service/GastoService.java

util/ConsolaUtil.java

Funcionalidades pendientes

Guardar información en archivo o base de datos
